# Rock Paper Scissors

A simple browser-based Rock Paper Scissor game built with HTML, CSS, and JavaScript.

## Features

- Choose Rock, Paper, or Scissors
- The computer makes a random move
- Displays game result instantly
- Tracks win, loss, and tie counts
- Reset button clears game statistics

## Files

- `index.html` - main page structure and game buttons
- `style.css` - styling and visual layout
- `script.js` - game logic, random computer choice, and score tracking

## How to use

1. Open (https://rock-paper-scissor-hhjh.onrender.com) in a web browser.
2. Click one of the buttons: Rock, Paper, or Scissors.
3. View the game result and stats below.
4. Click `Reset` to start fresh.

## Notes

- The game uses a 50/50/50 random selection split for the computer's move.
- Stats update after each play.
- The app is self-contained and requires no build tools.
